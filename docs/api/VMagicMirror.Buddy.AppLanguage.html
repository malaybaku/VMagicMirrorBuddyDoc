Remarks

VMagicMirrorのローカライズシステムの実装都合により、日本語・英語以外の言語は Unknown として扱われます。

多言語に詳細に対応できるサブキャラを作る場合、 IProperty を通じて言語選択UIを提供することも検討してください。
